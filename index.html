PROJECTS

Spotify Recommender System

A data set containing 100,000 Spotify tracks and their metadata was preprocessed using Pandas library in Python. Scikit-learn library was used to transform text track metadata into a vectorized data frame using TF-IDF methodology. Spotify's web API was accessed through Spotipy library to create a personal playlist vector to compare with each song in the vectorized dataset using cosine similarity. Track suggestions from this system and Spotify’s recommendation system often aligned, featuring the same songs or new songs by the artists already present in the personal playlist.

Topic Modeling: Latent Dirichlet Allocation (LDA)

Employed Natural Language Toolkit (NLTK) to process and clean unstructured text data from a dataset comprising 4540 Facebook statuses for analysis. Utilized Python's Scikit-learn packages for LDA topic modeling to identify the ten most prevalent topics within the Facebook statuses. Assigned a topic to each status using the probability distribution of topics. Grouped by topic, the sum of different Facebook status reactions was analyzed to gain a comprehensive summary of user sentiment regarding each topic.
