PROJECTS

Spotify Recommender System

A data set containing 1.1 million Spotify tracks and their metadata was preprocessed using Pandas library in Python. One-hot encoding was used to transform text track metadata (genre) to create a vectorized data frame. Spotify's web API was accessed through Spotipy library to acess user playlists and tracks to create a vector to compare with each song in the Spotify dataset using cosine similarity. Streamlit UI was utilized to enhance user experience and visualize track selection and generated recommendations. Achieved an average R-precision score of 32%, indicating the system's effectiveness in suggesting relevant items to users

Topic Modeling: Latent Dirichlet Allocation (LDA)

Employed Natural Language Toolkit (NLTK) to process and clean unstructured text data from a dataset comprising 4540 Facebook statuses for analysis. Utilized Python's Scikit-learn packages for LDA topic modeling to identify the ten most prevalent topics within the Facebook statuses. Assigned a topic to each status using the probability distribution of topics. Grouped by topic, the sum of different Facebook status reactions was analyzed to gain a comprehensive summary of user sentiment regarding each topic.
